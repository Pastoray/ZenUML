# ZenUML System Design Examples

> Note: This entire README.md document was also generated by the main contributor to this project, Rida.

This repository hosts a collection of sequence diagrams and system design patterns created using **ZenUML**.

The project serves as a reference for modeling various business processes, network protocols, and software architectures using the Diagram-as-Code approach.

## ğŸ› ï¸ How to Use

### Viewing Diagrams

* **Quick View:** Open the `.png` file in any image viewer or browser.

* **Edit/Analyze:** Open the `.zenuml` file. You can render it using:

  * The [ZenUML](https://app.zenuml.com/) Online Editor.

  * The **ZenUML Extension** for VS Code or IntelliJ IDEA.

### For New Contributors

1. Create a directory with your name under `examples/`.

2. Create a new folder for your example (e.g., `example1`).

3. Include the `.zenuml` source, a generated `.png`, and the reference `.java` sketch.

4. Update this README
